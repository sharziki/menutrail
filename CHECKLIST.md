## Features

- [ ] **1. Multi-photo menu items** - Upload multiple photos for hover animations
- [ ] **2. Custom CTA per restaurant** - Add/customize call-to-action for each restaurant
- [ ] **3. Multiple menus support** - Like fresh fish marketplace (multiple menu sections)
- [ ] **4. Address autopopulate API** - Integrate address autocomplete in onboarding
- [ ] **5. Real dashboard data** - Replace demo data with real data on dashboard
- [ ] **6. VPS deployment** - Get server + everything live on VPS
- [ ] **7. Demo/sandbox mode** - Test DoorDash delivery orders in sandbox before going live
- [ ] **8. Go High Level integration** - Auto-create/update contacts on orders, map fields, add notes per order
- [ ] **9. *(skipped in original)*
- [ ] **10. Gift cards** - Allow customers to purchase gift cards
- [ ] **11. Documentation** - Detailed GitHub docs for Pakistani dev to make changes
- [ ] **12. Mobile 2x2 grid layout** - Sidebar/main layout on mobile like Sizzle Fish
- [ ] **13. Mobile modals fix** - Click item to bring up popup with modifiers, special requests
- [ ] **14. Category descriptions** - Fix the description display for categories
- [ ] **15. Mandatory modifier popup** - Can't add to cart without going through popup, mandatory modifiers (lettuce, pickle, tomato, onion, etc.)
- [ ] **16. Checkout upcharge popups** - Customize upcharges via popup (e.g., drink size selection)
- [ ] **17. Pickup order sound** - Make the pickup order sound 6 seconds
